# macUpdate

[![Build Status][travis-badge]][travis-link]
[![Slack Room][slack-badge]][slack-link]

update your mac

## Install

With [fisherman]

```
fisher publicarray/mac-update
```

## Usage

```fish
update [system|mas|brew|npm|yarn|pip|composer|apm|gem|dotfiles|all|packages] [--help]
```
